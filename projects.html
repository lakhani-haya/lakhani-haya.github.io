Projects

Spotify Data Analysis and Visualization Dashboard

For this project, I created a multi-dashboard Tableau visualization to analyze and visualize Spotify song data. The dashboards explored various song statistics, popularity trends, and other musical elements. The goal was to provide a visual representation of song features, track performance, and listener preferences based on key musical attributes like danceability, energy, and acousticness. This allowed users to understand how different variables impact a song's success on the platform.

Spotify Song Recommendation System

I developed an application using Python and the Spotify API to generate highly specific song recommendations for users. By asking users a series of questions about their musical preferences—such as preferred genre, release timeframes, mood, and other song characteristics—the application suggests personalized playlists. The tool uses real-time data from Spotify's music database to recommend songs that match the user's unique preferences, offering a tailored listening experience.

Database and RMDBS Development for a Business

For a course project, I established a fully functional database and relational database management system (RDBMS) for a fictional company. The project involved designing the database structure, defining schemas, and ensuring the system aligned with the company’s operating models, preferences, and business needs. I implemented normalization techniques and optimized queries to ensure data integrity, security, and efficiency, ultimately resulting in a robust backend system suitable for a real-world business environment.
